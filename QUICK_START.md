## ðŸ†˜ Troubleshooting

### "Can't reach this page" or "Connection refused"
**Solution:**
1. Make sure computer is running
2. Check WiFi is on
3. Try: http://172.30.224.1:3000
4. Hard refresh page (pull down on mobile)
5. Wait 10-15 seconds to load

### "Page is blank"
**Solution:**
1. Clear browser cache
2. Close and reopen browser
3. Try a different browser
4. Check WiFi connection
5. Restart phone WiFi

### "Features not showing"
**Solution:**
1. Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
2. Clear cache
3. Make sure you logged in (even as guest)
4. Wait for complete page load

### Voice chat not working
**Solution:**
1. Allow microphone permission
2. Check browser microphone access in settings
3. Try text chat instead
4. Use different browser (Chrome best for voice)
